# yuml
Lightweight UML based MBSE application built on yaml syntax UML config files

## Currently Implemented

### yaml parsing
Currently Implemented
    -Element
        -id and child
    -NamedElement
        -name
    -Classifier
        -attributes list
    -Property
        -type (started default values)
    -Type
    -Class
    -PrimitveType (not parsed)
        -created enum
    -ValueSpecification (not parsed)
    -Namespace
    -Model
    -Literals
    -Operations
    -OpaqueBehaviors
